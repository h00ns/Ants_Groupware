# 🐜비대면 협업을 위한 그룹웨어 서비스 Ants

<br>

# 프로젝트 소개
 - **Ants**는 비대면 협업과 효율적인 업무 수행을 지원하는 그룹웨어 웹 서비스입니다.
 - 개발 기간 : 2021.07.05 ~ 2021.08.20 (7주)
 - 삼성 청년 SW 아카데미 2학기 공통프로젝트 대전 1반 최우수상 수상🏆

<br>

# 👩 팀원 역할
| 팀원 | 역할 | 담당 업무
| ------ | ------ | ------ |
| 정범진 | 팀장 | BE 개발, 팀장 |
| 이재영 | 팀원 | BE 테크리더 |
| 김지안 | 팀원 | FE 개발, 기획 |
| 황지훈 | 팀원 | FE 테크리더 |
